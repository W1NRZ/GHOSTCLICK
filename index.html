GhostClick

simple auto clicker for fast automation
version 1.0 • windows executable (.exe)
download latest version
how to use:
  • download the file above
  • open the .exe (no python needed)
  • set cps + mouse button
  • press apply
  • F6 = start / stop clicking
  • F7 = exit app
⚠ no installation required. just run the .exe file directly.
features:
  • fast click automation (custom cps)
  • left / right / middle mouse support
  • hotkey control (f6, f7, f8)
  • lightweight + low cpu usage
